# 🌐 OSI Model (Open Systems Interconnection Model)

The **OSI (Open Systems Interconnection) Model** is a conceptual framework used to understand how data travels from one device to another over a network.

It divides the communication process into **7 layers**, where each layer performs a specific function.

The OSI model helps developers, network engineers, and system administrators understand, troubleshoot, and design network communication efficiently.
