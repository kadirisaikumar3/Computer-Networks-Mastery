# 🚀 Chapter 2 – OSI Model

## 📖 Overview

The **OSI (Open Systems Interconnection) Model** is a conceptual framework that explains how data travels between devices over a network.

Although the Internet uses the TCP/IP model, the OSI model is widely used for learning, troubleshooting, and interview preparation.
