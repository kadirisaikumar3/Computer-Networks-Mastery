# Data Link Layer

## Overview

The **Data Link Layer** is **Layer 2** of the OSI Model.

It provides reliable communication between directly connected devices on the same network.

---

# Responsibilities

- Frame Creation
- MAC Addressing
- Error Detection
- Flow Control
- Access Control

---

# Data Unit

```
Frame
```

---

# Address Used

```
MAC Address
```

Example:

```
00:1A:2B:3C:4D:5E
```

---

# Devices

- Switch
- Bridge

---

# Sublayers

## LLC (Logical Link Control)

Controls communication between upper layers.

---

## MAC (Media Access Control)

Controls access to the transmission medium.

---

# Functions

- Frame Delivery
- Error Detection (CRC)
- MAC Addressing
- Local Network Communication
