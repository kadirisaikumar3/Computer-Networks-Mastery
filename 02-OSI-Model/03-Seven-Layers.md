# Layer Responsibilities

## Application

Provides services to applications.

Examples:

- HTTP
- FTP
- SMTP
- DNS

---

## Presentation

- Encryption
- Compression
- Translation

---

## Session

Creates and manages communication sessions.

---

## Transport

Provides:

- Reliability
- Error Recovery
- Flow Control

Protocol:

- TCP
- UDP

---

## Network

Responsible for:

- IP Addressing
- Routing

Device:

Router

---

## Data Link

Responsible for:

- MAC Address
- Frame Transmission

Device:

Switch

---

## Physical

Responsible for transmitting raw bits.

Examples:

- Cable
- Fiber
- Radio Signals
