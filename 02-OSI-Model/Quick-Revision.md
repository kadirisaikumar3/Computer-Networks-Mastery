# OSI Model Quick Revision

| Layer | Function | Protocols | Device | PDU |
|--------|----------|-----------|---------|-----|
| Application | User Services | HTTP, HTTPS, FTP | - | Data |
| Presentation | Encryption | SSL/TLS | - | Data |
| Session | Session Management | NetBIOS | - | Data |
| Transport | Reliable Delivery | TCP, UDP | - | Segment |
| Network | Routing | IP, ICMP | Router | Packet |
| Data Link | MAC Addressing | Ethernet | Switch | Frame |
| Physical | Transmission | Cable | Hub | Bits |
