# Network Layer

## Overview

The **Network Layer** is **Layer 3** of the OSI Model.

It is responsible for logical addressing and routing packets between different networks.

---

# Responsibilities

- Logical Addressing
- Routing
- Packet Forwarding
- Path Selection
- Fragmentation

---
