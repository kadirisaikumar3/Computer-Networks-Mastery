# Google Interview Corner

This section contains the most frequently asked OSI Model interview questions.

---

# Basic Questions

## What is the OSI Model?

A conceptual framework that divides network communication into seven layers.

---

## Why was the OSI Model created?

To standardize communication between different systems.

---

## How many layers are there?

```
7
```

---

## Which model is actually used on the Internet?

TCP/IP Model.

---

# Layer-Based Questions

## Which layer transmits bits?

Physical Layer.

---

## Which layer uses MAC addresses?

Data Link Layer.

---

## Which layer performs routing?

Network Layer.

---

## Which layer provides reliable communication?

Transport Layer.

---

## Which layer manages sessions?

Session Layer.

---

## Which layer performs encryption?

Presentation Layer.

---

## Which layer interacts with users?

Application Layer.

---

# Scenario-Based Questions

## A cable is damaged. Which layer is affected?

Physical Layer.

---

## A router cannot forward packets. Which layer is affected?

Network Layer.

---

## HTTPS data is encrypted. Which layer performs encryption?

Presentation Layer.

---

## TCP connection fails. Which layer is responsible?

Transport Layer.
