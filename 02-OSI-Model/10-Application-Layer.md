# Application Layer

## Overview

The **Application Layer** is **Layer 7** of the OSI Model.

It is the layer closest to the end user and provides network services directly to applications.

It acts as the interface between the user and the network.

---

# Responsibilities

- Provides network services to applications
- File transfer
- Email communication
- Web browsing
- Remote login
- Name resolution
