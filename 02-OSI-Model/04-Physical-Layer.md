# Physical Layer

## Overview

The **Physical Layer** is the **first layer (Layer 1)** of the OSI Model.

It is responsible for transmitting raw binary bits (0s and 1s) over a physical communication medium.

---

# Responsibilities

- Transmits raw bits
- Defines cables and connectors
- Defines voltage levels
- Defines data transmission speed
- Defines signal timing
