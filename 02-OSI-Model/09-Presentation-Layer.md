# Presentation Layer

## Overview

The **Presentation Layer** is **Layer 6** of the OSI Model.

It is responsible for **data formatting**, **translation**, **encryption**, and **compression**.

---

# Responsibilities

- Data Translation
- Data Encryption
- Data Decryption
- Data Compression
- Data Decompression

---

# Data Unit

```
Data
```

---

# Functions

## Translation

Converts data into a common format.

---

## Encryption

Protects data during transmission.

Example:

```
Plain Text

↓

Encryption

↓

Cipher Text
```

---

## Decryption

Converts encrypted data back to its original form.

---

## Compression

Reduces data size before transmission.
