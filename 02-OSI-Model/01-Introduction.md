# Introduction

## What is the OSI Model?

The **OSI (Open Systems Interconnection) Model** is a conceptual framework that explains how data travels between two devices over a network.

It divides network communication into **7 layers**, where each layer performs a specific function.

---

# Why was the OSI Model Created?

Before the OSI model, different vendors used different communication standards.

The OSI model introduced a common framework so that devices from different manufacturers could communicate with each other.

---

# Goals of the OSI Model

- Standardize network communication
- Improve interoperability
- Simplify troubleshooting
- Divide complex communication into smaller layers
- Allow independent development of protocols

---

# Characteristics

- Contains **7 Layers**
- Each layer has a specific responsibility
- Each layer communicates with the layer above and below it
- Supports modular network design

---

# Advantages

- Easy to understand
- Modular architecture
- Eases troubleshooting
- Standard reference model
- Independent layer development

---

# Limitations

- Mainly a reference model
- Internet actually uses TCP/IP
- Slightly more complex than TCP/IP
