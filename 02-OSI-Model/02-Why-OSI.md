# Why OSI Model?

## Why Do We Need the OSI Model?

The OSI Model was created to make communication between different computer systems easier and standardized.

Without a common model, every manufacturer would use its own networking rules.

---

# Problems Before OSI

- Different hardware vendors
- Different communication protocols
- Compatibility issues
- Difficult troubleshooting
- No common standard

---

# Benefits of the OSI Model

## Standardization

Provides a common networking standard.

---

## Interoperability

Allows devices from different vendors to communicate.

---

## Layered Design

Each layer performs a separate task.

This makes development and maintenance easier.

---

## Easier Troubleshooting

Problems can be isolated to a specific layer.

Example:

```
No Cable

↓

Physical Layer Problem
```

---

## Flexibility

Changes in one layer usually do not affect the others.

---

# Example

When opening a website:

```
Browser

↓

↓

Transport Layer

↓

Network Layer

↓

Data Link Layer

↓

```

Each layer performs its own job.
