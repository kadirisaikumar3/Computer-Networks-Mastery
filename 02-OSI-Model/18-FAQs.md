# Frequently Asked Questions (FAQs)

## What is the OSI Model?

A conceptual framework that divides network communication into seven layers.

---

## How many layers are in the OSI Model?

```
7
```

---

## Which model is actually used on the Internet?

TCP/IP Model.

---

## Which layer uses MAC Addresses?

Data Link Layer.

---

## Which layer uses IP Addresses?

Network Layer.

---

## Which layer provides reliable communication?

Transport Layer.

---

## Which layer performs routing?

Network Layer.

---

## Which layer performs encryption?

Presentation Layer.

---

## Which layer interacts with the user?

Application Layer.

---

## Which device works at Layer 2?

Switch.

---

## Which device works at Layer 3?

Router.

---

## Which layer transmits bits?

Physical Layer.
