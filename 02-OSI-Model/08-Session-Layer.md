# Session Layer

## Overview

The **Session Layer** is **Layer 5** of the OSI Model.

It establishes, manages, and terminates communication sessions between applications.

---

# Responsibilities

- Session Establishment
- Session Maintenance
- Session Termination
- Synchronization
- Checkpointing

---

# Data Unit

```
Data
```

---

# Functions

- Opens communication sessions
- Keeps sessions active
- Closes completed sessions
- Handles session recovery
