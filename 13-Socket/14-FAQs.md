# Frequently Asked Questions (FAQs)

## 1. What is a Socket?

An endpoint of communication between two applications.

---

## 2. What identifies a Socket?

- IP Address
- Port Number
- Protocol

---

## 3. Which Java package contains socket classes?

```java
java.net
```

---

## 4. What is ServerSocket?

A class used by the server to listen for incoming client connections.

---

## 5. What is Socket?

A class used by the client to connect to a server.

---

## 6. Which method waits for a client?

```java
accept()
```

---

## 7. Which method closes a socket?

```java
close()
```

---

## 8. Can multiple clients connect simultaneously?

Yes.

Using multithreading.

---

## 9. Which protocol do browsers use?

TCP.

---

## 10. Which protocol do online games use?

Mostly UDP.

---

## 11. Which class creates threads?

```java
Thread
```

---

## 12. Why is Socket Programming important?

Because it enables communication between distributed applications.
