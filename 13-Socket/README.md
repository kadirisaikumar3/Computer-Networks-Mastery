# 🌐 Chapter 13 – Socket Programming

## 📖 Overview

**Socket Programming** enables applications running on different computers (or on the same computer) to communicate over a network using protocols such as **TCP** and **UDP**.

Sockets are the foundation of many modern applications, including:

- Web Browsers
- Chat Applications
- Email Systems
- Online Games
- Video Streaming
- Banking Applications
