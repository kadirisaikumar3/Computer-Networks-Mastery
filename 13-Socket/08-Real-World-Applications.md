# Real-World Applications

## Socket Programming in Real Life

Sockets are used in almost every network application.

---

# 1. Web Browsers

```
Chrome

↓

Socket

↓

Google Server

↓

HTML Response
```

Uses:

TCP Socket

---

# 2. Chat Applications

Examples:

- WhatsApp
- Telegram
- Slack
- Discord

Messages travel through sockets.

---

# 3. Online Banking

```
Customer

↓

Socket

↓

Bank Server

↓

Database

↓

Response
```

Reliable TCP communication is essential.

---

# 4. Email

Protocols:

- SMTP
- POP3
- IMAP

All use TCP sockets.

---

# 5. Multiplayer Games

```
Player

↓

Game Socket

↓

Game Server

↓

Other Players
```

Mostly UDP sockets.

---

# 6. Video Streaming

Examples:

- Netflix
- YouTube
- Twitch

Uses both TCP and UDP depending on the service.

---

# 7. Video Calls

Examples:

- Zoom
- Google Meet
- Microsoft Teams

Low latency is achieved using UDP sockets for media.

---

# 8. IoT Devices

Smart devices communicate using sockets.

Examples:

- Smart Bulbs
- Smart Cameras
- Smart Watches

---

# 9. Cloud Services

Examples:

- AWS
- Azure
- Google Cloud

Millions of socket connections are handled every second.
