# Network Devices

## What are Network Devices?

Network devices are hardware components that enable communication between computers and other devices in a network.

---

# Common Network Devices

- Hub
- Switch
- Router
- Modem
- Access Point
- Bridge
- Repeater
- Gateway
- Firewall

---

# 1. Hub

- Layer 1 Device
- Broadcasts data to all devices
- No MAC Address Table

---

# 2. Switch

- Layer 2 Device
- Uses MAC Addresses
- Sends data only to the destination device

---

# 3. Router

- Layer 3 Device
- Uses IP Addresses
- Connects different networks

---

# 4. Modem

Converts:

```
Digital

↓

Analog

↓

Digital
```

Used for Internet connectivity.

---

# 5. Access Point (AP)

Provides wireless connectivity to devices.

Examples:

- Home Wi-Fi
- Office Wi-Fi

---

# 6. Bridge

Connects two LAN segments.

Works on:

```
Layer 2
```

---

# 7. Repeater

Regenerates weak network signals to extend transmission distance.

Works on:

```
Layer 1
```

---

# 8. Gateway

Acts as an entry and exit point between different networks.

---

# 9. Firewall

Filters incoming and outgoing network traffic.

Improves network security.

---

# Device Summary

| Device | OSI Layer | Function |
|----------|-----------|----------|
| Hub | Layer 1 | Broadcast |
| Repeater | Layer 1 | Signal Regeneration |
| Bridge | Layer 2 | Connect LAN Segments |
| Switch | Layer 2 | MAC-Based Forwarding |
| Router | Layer 3 | IP Routing |
| Gateway | Multiple | Connect Different Networks |
| Firewall | Multiple | Security |
| Access Point | Layer 2 | Wireless Access |
| Modem | Layer 1 | Internet Connectivity |

---

# Google Interview Questions

## Difference between Hub and Switch?

Hub broadcasts to all devices.

Switch forwards data using MAC addresses.

---

## Difference between Switch and Router?

Switch connects devices in a LAN.

Router connects different networks.

---

## Which device provides Wi-Fi?

Access Point.
