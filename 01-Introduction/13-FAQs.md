# Frequently Asked Questions (FAQs)

## What is a Computer Network?

A collection of interconnected devices that communicate with each other.

---

## Which topology is most commonly used?

Star Topology.

---

## Which topology is the most reliable?

Mesh Topology.

---

## Which network covers a building?

LAN.

---

## Which network covers the world?

WAN.

---

## Which device connects different networks?

Router.

---

## Which device provides Wi-Fi?

Access Point.

---

## Which device filters traffic?

Firewall.

---

## What is a protocol?

A set of communication rules between devices.

---

## Which architecture is used by web applications?

Client-Server Architecture.
