# Introduction

## What is a Computer Network?

A **Computer Network** is a collection of two or more devices connected together to exchange data and share resources.

These devices communicate using standard networking protocols such as TCP/IP.

---

# Why Do We Need Computer Networks?

Computer Networks make it possible to:

- Share files
- Share printers
- Access the Internet
- Send emails
- Make video calls
- Build distributed applications
- Enable cloud computing

---

# Components of a Network

A basic computer network consists of:

- Computers
- Servers
- Routers
- Switches
- Cables or Wireless Media
- Network Interface Cards (NICs)

---

# Network Communication

```
Computer A

↓

Network

↓

Computer B
```

Information is divided into packets before transmission.

# Advantages

- Resource Sharing
