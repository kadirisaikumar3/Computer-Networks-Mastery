# Real-World Examples

## Example 1 – Home Wi-Fi Network

### Scenario

A family has:

- 3 Mobile Phones
- 2 Laptops
- 1 Smart TV
- 1 Printer

All devices connect to a Wi-Fi router.

```
           Internet
               │
            Router
         ┌─────┼─────┐
         │     │     │
     Laptop  Mobile  TV
         │
      Printer
```

### Network Type

LAN (Local Area Network)

---

## Example 2 – College Computer Lab

### Scenario

A college lab contains 50 computers connected through switches.

```
          Switch
     ┌────┼────┐
     │    │    │
    PC   PC   PC
     │    │    │
   (More Computers)
```

### Advantages

- Centralized Management
- Easy File Sharing
- Internet Access

## Example 3 – Banking Network

```

Internet

↓

Router

Branch B
```
