# Computer Networks

## How Does a Computer Network Work?

When one device wants to communicate with another:

```
Sender

↓

Packet Creation

↓

Transmission Medium

↓

Router/Switch

↓

Receiver
```

Data is divided into packets before transmission.

---

# Basic Network Components

## Sender

Device that sends data.

Examples:

- Laptop
- Mobile
- Server

---

## Receiver

Device that receives data.

---

## Transmission Medium

Carries data.

Examples:

- Ethernet Cable
- Fiber Optic Cable
- Wi-Fi

---

## Protocol

Rules that govern communication.

Examples:

- TCP
- UDP
- HTTP
- HTTPS
- DNS

---

# Types of Communication

- Wireless
