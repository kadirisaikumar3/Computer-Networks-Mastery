# 🌐 Chapter 9 – Dynamic Host Configuration Protocol (DHCP)

## 📖 Overview

**Dynamic Host Configuration Protocol (DHCP)** is a network management protocol that automatically assigns IP configuration to devices on a network.

Instead of manually configuring every device with an IP Address, Subnet Mask, Default Gateway, and DNS Server, DHCP performs this task automatically.
