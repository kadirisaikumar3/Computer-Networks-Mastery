# 🌐 Chapter 11 – TCP (Transmission Control Protocol)

## 📖 Overview

**TCP (Transmission Control Protocol)** is a connection-oriented Transport Layer protocol that provides reliable, ordered, and error-checked delivery of data between applications.

TCP is used whenever reliability is important, such as:

- Web Browsing (HTTP/HTTPS)
- Email
- File Transfer
- Online Banking
- Cloud Applications
