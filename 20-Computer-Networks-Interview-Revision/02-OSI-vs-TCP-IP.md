# OSI Model vs TCP/IP Model

## Overview

The **OSI Model** is a theoretical reference model developed to standardize network communication.

The **TCP/IP Model** is the practical networking model used on the Internet.

---

# OSI Model (7 Layers)

| Layer | Name |
|--------|------|
| 7 | Application |
| 6 | Presentation |
| 5 | Session |
| 4 | Transport |
| 3 | Network |
| 2 | Data Link |
| 1 | Physical |

---

# TCP/IP Model (4 Layers)

| Layer | Name |
|--------|------|
| 4 | Application |
| 3 | Transport |
| 2 | Internet |
| 1 | Network Access |

---

# Layer Mapping

| OSI Model | TCP/IP Model |
|------------|--------------|
| Application | Application |
| Presentation | Application |
| Session | Application |
| Transport | Transport |
| Network | Internet |
| Data Link | Network Access |
| Physical | Network Access |

---

# Differences

| OSI | TCP/IP |
|------|---------|
| 7 Layers | 4 Layers |
| Reference Model | Protocol Suite |
| Theoretical | Practical |
| Developed by ISO | Developed by DoD |
| Protocol Independent | Protocol Based |

---

# Advantages of OSI

- Easy to understand
- Modular architecture
- Standard reference model
- Useful for learning

---

# Advantages of TCP/IP

- Used on the Internet
- Highly scalable
- Reliable
- Supports routing
