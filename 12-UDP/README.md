# 🌐 Chapter 12 – UDP (User Datagram Protocol)

## 📖 Overview

**UDP (User Datagram Protocol)** is a **connectionless Transport Layer protocol** designed for fast communication with minimal overhead.

Unlike TCP, UDP does not establish a connection before sending data. It does not guarantee delivery, ordering, or retransmission, making it suitable for real-time applications where speed is more important than reliability.
