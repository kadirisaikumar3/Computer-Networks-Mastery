# BGP (Border Gateway Protocol)

## What is BGP?

BGP (Border Gateway Protocol) is the routing protocol that connects different Autonomous Systems (AS) on the Internet.

It is known as the **Internet Routing Protocol**.

---

# Characteristics

- Path Vector Protocol
- Exterior Gateway Protocol (EGP)
- Highly Scalable
- Policy-Based Routing
- Used by Internet Service Providers

---

# Where is BGP Used?

- Internet
- ISPs
- Cloud Providers
- Large Data Centers

---

# How BGP Works

1. Routers establish BGP neighbors.
2. They exchange network routes.
3. Multiple paths are compared.
4. Best path is selected.
5. Routing table is updated.

---

# Path Selection Factors

- AS Path
- Local Preference
- MED (Multi-Exit Discriminator)
- Next Hop

---

# Advantages

- Supports the entire Internet
- Highly scalable
- Flexible routing policies
- Reliable

---

# Disadvantages

- Complex configuration
- Slow convergence
- Requires careful planning

---

When you open:

Your ISP uses BGP to determine the best route to Google's network.
