# Quick Revision

## Routing in One Line

Routing is the process of forwarding packets between different networks using routers.

---

# Routing Flow

```
Source

↓

Router

↓

Routing Table

↓

Best Route

↓

Destination
```

---

# Router's Job

- Receive Packet
- Check Destination IP
- Search Routing Table
- Select Best Route
- Forward Packet

---

# Static vs Dynamic Routing

| Static | Dynamic |
|---------|----------|
| Manual | Automatic |
| Small Networks | Large Networks |
| Fixed Routes | Learns Routes |

---

# Routing Protocols

| Protocol | Type | Metric |
|----------|------|--------|
| RIP | Distance Vector | Hop Count |
| OSPF | Link-State | Cost |
| BGP | Path Vector | Path Attributes |
