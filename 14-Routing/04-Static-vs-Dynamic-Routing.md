# Static Routing vs Dynamic Routing

## Overview

Routing can be configured in two ways:

- Static Routing
- Dynamic Routing

Both help routers determine the best path to forward packets.

---

# Static Routing

## What is Static Routing?

Static Routing is a routing method where routes are manually configured by the network administrator.

The router does not automatically update routes.

---

# Characteristics

- Manual Configuration
- Fixed Routes
- Simple
- No Automatic Updates
- Low CPU Usage

---

# Advantages

- Simple to configure
- High security
- Predictable routing
- Low resource usage

---

# Disadvantages

- Difficult to maintain in large networks
- Manual updates required
- Not scalable
- No automatic failure recovery

---

# Dynamic Routing

## What is Dynamic Routing?

Dynamic Routing automatically learns and updates routes using routing protocols.

Routers exchange routing information with each other.

---

# Characteristics

- Automatic Updates
- Learns new routes
- Scalable
- Adapts to failures
- Uses Routing Protocols

---

# Advantages

- Easy to manage
- Automatically adapts to network changes
- Suitable for large networks
- Better fault tolerance

---

# Disadvantages

- Higher CPU and memory usage
- More complex configuration
- Additional network traffic

---

# Comparison

| Feature | Static Routing | Dynamic Routing |
|----------|----------------|-----------------|
| Configuration | Manual | Automatic |
| Updates | Manual | Automatic |
| Scalability | Low | High |
| Maintenance | Difficult | Easy |
| CPU Usage | Low | Higher |
| Best For | Small Networks | Large Networks |
