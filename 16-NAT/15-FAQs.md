# Frequently Asked Questions (FAQs)

## 1. What is NAT?

Network Address Translation translates private IP addresses into public IP addresses.

---

## 2. Why is NAT needed?

To conserve IPv4 addresses and allow Internet access.

---

## 3. Which device performs NAT?

Router or Firewall.

---

## 4. What are the three types of NAT?

- Static NAT
- Dynamic NAT
- PAT

---

## 5. Which NAT type is most common?

PAT.

---

## 6. What is NAT Overload?

Another name for PAT.

---

## 7. Which NAT type is used for servers?

Static NAT.

---

## 8. What is a NAT Translation Table?

A table that stores mappings between private and public addresses.

---

## 9. Does NAT improve security?

It hides private IP addresses, but it is not a complete security solution.

---

## 10. Can multiple devices share one public IP?

Yes, using PAT.
