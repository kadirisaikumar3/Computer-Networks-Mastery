# 🤝 Contributing to Computer Networks Mastery

Thank you for your interest in contributing to **Computer Networks Mastery**! 🎉

We welcome contributions that improve the quality of this repository and help the learning community.

---

# 📋 How to Contribute

You can contribute by:

- 📝 Fixing spelling or grammar mistakes
- 📚 Improving explanations
- 💡 Adding real-world examples
- 🧪 Adding hands-on labs
- ❓ Adding interview questions
- 🧠 Improving cheat sheets and mind maps
- 🐛 Reporting issues
- 🚀 Suggesting new networking topics

---

# 🚀 Getting Started

## 1. Fork the Repository

Click the **Fork** button on GitHub.

---

## 2. Clone the Repository

```bash
git clone https://github.com/kadirisaikumar3/Computer-Networks-Mastery.git
```

---

## 3. Create a New Branch

```bash
git checkout -b feature/your-feature-name
```

Example:

```bash
git checkout -b feature/add-dns-examples
```

---

## 4. Make Your Changes

Keep your changes:

- Well organized
- Easy to understand
- Properly formatted
- Grammatically correct

---

## 5. Commit Your Changes

```bash
git add .

git commit -m "Add DNS real-world examples"
```

---

## 6. Push the Branch

```bash
git push origin feature/add-dns-examples
```

---

## 7. Create a Pull Request

Open a Pull Request describing:

- What you changed
- Why you changed it
- Any screenshots (if applicable)

---

# 📚 Contribution Guidelines

Please ensure that your contribution:

- Follows Markdown formatting
- Uses simple and clear explanations
- Includes practical examples when possible
- Maintains the repository structure
- Does not remove existing learning material without reason

---

# 💬 Reporting Issues

If you find:

- Incorrect information
- Broken links
- Formatting issues
- Missing topics

Please create a GitHub Issue.

---

# ❤️ Thank You

Every contribution, no matter how small, helps improve this repository for learners around the world.

Happy Learning! 🚀
