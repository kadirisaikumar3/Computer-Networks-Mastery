# Contributing

Thank you for your interest in contributing to **Computer Networks Mastery**.

## Repository Guidelines

- Follow the existing folder structure.
- Write clear and beginner-friendly explanations.
- Add diagrams wherever possible.
- Include real-world examples.
- Add interview questions for every topic.
- Use Markdown formatting consistently.

## Commit Messages

Examples:

```
Add ARP chapter
```

```
Improve DNS documentation
```

```
Fix subnetting example
```

## Pull Requests

Before submitting a pull request:

- Check formatting.
- Verify links.
- Test code examples.
- Update documentation if required.

Thank you for helping improve this repository.
