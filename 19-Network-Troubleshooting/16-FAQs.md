# Frequently Asked Questions (FAQs)

## 1. What is Network Troubleshooting?

The process of identifying and resolving network problems.

---

## 2. What command checks connectivity?

```cmd
ping
```

---

## 3. What command checks DNS?

```cmd
nslookup
```

---

## 4. What command shows IP configuration?

```cmd
ipconfig /all
```

---

## 5. What command displays active connections?

```cmd
netstat -an
```

---

## 6. What command displays the routing table?

```cmd
route print
```

---

## 7. What command displays ARP entries?

```cmd
arp -a
```

---

## 8. Why would ping 8.8.8.8 work but google.com fail?

DNS resolution failure.

---

## 9. What causes packet loss?

- Congestion
- Faulty Hardware
- Weak Wi-Fi

---

## 10. What causes high latency?

- Congestion
- Long Distance
- Network Overload

---
