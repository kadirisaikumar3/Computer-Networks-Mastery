# Common Network Problems

## Overview

Network issues can occur due to hardware failures, software misconfigurations, or external factors.

Understanding common problems helps diagnose issues quickly.

---

# 1. No Internet Connection

Possible Causes:

- Router Failure
- ISP Outage
- Loose Cable
- Incorrect IP Configuration

---

# 2. Slow Internet

Possible Causes:

- High Network Traffic
- Low Bandwidth
- Weak Wi-Fi Signal
- Malware

---

# 3. DNS Resolution Failure

Symptoms:

- Websites don't open by name.
- Websites open using IP addresses.

Possible Cause:

DNS Server Issue.

---

# 4. IP Address Conflict

Two devices have the same IP address.

Result:

Network connectivity problems.

---

# 5. Packet Loss

Symptoms:

- Slow applications
- Video call interruptions
- Timeouts

Possible Causes:

- Congestion
- Faulty Cable
- Bad Wi-Fi Signal

---

# 6. High Latency

Symptoms:

- Gaming lag
- Slow responses
- Delayed communication

---

# 7. Routing Problems

Symptoms:

- Cannot reach remote networks.
- Packets take incorrect paths.

---

# 8. Firewall Blocking Traffic

Firewall rules may block:

- Applications
- Ports
- Services

---

# Symptoms and Causes

| Problem | Possible Cause |
|----------|----------------|
| No Internet | Router / ISP |
| Website Not Opening | DNS |
| Slow Network | Congestion |
| High Ping | Latency |
| Connection Drops | Packet Loss |
| No Remote Access | Routing |
| App Blocked | Firewall |

---

# Google Interview Questions

## What causes packet loss?

- Congestion
- Faulty hardware
- Weak wireless signal

---

## What is an IP conflict?

Two devices using the same IP address.

---

## Why would a website open using its IP but not its name?

DNS resolution failure.
