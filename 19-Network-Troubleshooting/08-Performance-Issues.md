# Performance Issues

## What are Performance Issues?

Performance issues occur when the network works but responds slowly or inconsistently.

---

# Common Symptoms

- Slow Internet
- High Latency
- Packet Loss
- Slow Downloads
- Video Buffering
- Voice Call Lag

---

# Common Causes

- Network Congestion
- Low Bandwidth
- Weak Wi-Fi Signal
- Faulty Hardware
- Malware
- High CPU Usage
- Background Downloads

---

# Performance Metrics

## Latency

Time taken for data to travel.

Measured in:

```
Milliseconds (ms)
```

Lower latency is better.

---

## Bandwidth

Maximum amount of data that can be transmitted.

Measured in:

```
Mbps

Gbps
```

---

## Packet Loss

Packets lost during transmission.

Ideal:

```
0%
```

---

## Jitter

Variation in packet arrival time.

Important for:

- Video Calls
- Voice Calls
- Online Gaming

---

# Useful Commands

Check latency:

```cmd
ping google.com
```

Trace route:

```cmd
tracert google.com
```

View connections:

```cmd
netstat -an
```

---

# How to Improve Performance

- Restart Router
- Upgrade Bandwidth
- Use Ethernet
- Reduce Network Traffic
- Close Background Downloads
- Update Network Drivers
