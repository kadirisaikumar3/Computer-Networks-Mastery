# Why Firewalls are Needed

## The Security Problem

Every computer connected to the Internet is exposed to potential threats.

Examples:

- Hackers
- Malware
- Viruses
- Ransomware
- Unauthorized Access

Without protection, these threats can compromise systems.

---

# How Firewalls Help

A firewall examines network traffic and decides whether to:

- Allow the traffic
- Block the traffic

based on predefined security rules.

---

# Benefits of Firewalls

- Prevent unauthorized access
- Block malicious traffic
- Protect sensitive data
- Control network access
- Monitor network activity

# Common Threats Blocked

- Port Scanning
- Unauthorized Login Attempts
- Malware Communication
- Suspicious Connections
- Unknown Services

---

# Real-World Example

An employee accesses a company website.

```
Employee

↓

Firewall

↓

Company Server
```

The firewall verifies that the connection follows company security rules.

# Google Interview Questions

## Why is a Firewall required?

To protect computers and networks from unauthorized access and malicious traffic.

---

## Does a Firewall inspect all traffic?

Yes, it inspects traffic according to configured rules.
