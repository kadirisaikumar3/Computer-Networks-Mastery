# Real-World Examples

## Example 1 – Home Network

```
Internet

↓

Wi-Fi Router
(Firewall Enabled)

↓

Laptop

Phone

Smart TV
```

The router's built-in firewall blocks unauthorized incoming traffic while allowing legitimate Internet access.
