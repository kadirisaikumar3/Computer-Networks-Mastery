# 🌐 Chapter 8 – Domain Name System (DNS)

## 📖 Overview

The **Domain Name System (DNS)** is often called the **Phonebook of the Internet**.

Humans prefer easy-to-remember domain names like:

```
www.google.com
```

Computers communicate using **IP Addresses**.

DNS translates a **domain name** into its corresponding **IP Address**, allowing users to access websites without remembering numerical addresses.
