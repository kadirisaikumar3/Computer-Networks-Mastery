# 🖥️ Chapter 7 – Address Resolution Protocol (ARP)

## 📖 Overview

Address Resolution Protocol (ARP) is a Layer 2 protocol used to map an IPv4 address to a MAC address within a local network.

Whenever one device wants to communicate with another device on the same LAN, it first needs to know the destination device's MAC address. ARP helps discover that MAC address using the destination IP address.
