# Common Mistakes

---

# Mistake 1

## Thinking Switches and Routers are the Same

❌ Wrong

Both perform the same function.

✅ Correct

Switch → Same Network

Router → Different Networks

---

# Mistake 2

## Confusing MAC and IP Address

❌ Wrong

Switch uses IP.

Router uses MAC.

✅ Correct

Switch → MAC Address

Router → IP Address

---

# Mistake 3

## Thinking Switch Operates on Layer 3

❌ Wrong

Switch works on Layer 3.

✅ Correct

Switch → Layer 2

Router → Layer 3

---

# Mistake 4

## Forgetting MAC Address Table

Switch always checks the MAC Address Table before forwarding a frame.

---

# Mistake 5

## Thinking Router Uses MAC Table

Routers use Routing Tables.

Switches use MAC Address Tables.

---

# Mistake 6

## Thinking VLANs Can Communicate Directly

Different VLANs require a Router or Layer 3 Switch for communication.

---

# Mistake 7

## Confusing Broadcast and Collision Domains

Router separates Broadcast Domains.

Switch creates separate Collision Domains.

---
