# Quick Revision

## Switch

- Layer 2 Device
- Uses MAC Address
- Connects devices in LAN
- Uses MAC Address Table
- Forwards Frames

---

## Router

- Layer 3 Device
- Uses IP Address
- Connects Networks
- Uses Routing Table
- Forwards Packets

---
