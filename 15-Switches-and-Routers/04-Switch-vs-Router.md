# Router

## What is a Router?

A **Router** is a networking device that connects different networks and forwards packets based on **IP Addresses**.

It enables communication between LANs and the Internet.

---

# Main Functions

- Connects Networks
- Routes Packets
- Maintains Routing Table
- Selects Best Path

---

# How a Router Works

```
Laptop

↓

Router

↓

Internet

↓

Server
```

The router receives a packet.

↓

Checks the destination IP Address.

↓

Searches the Routing Table.

↓

Selects the best route.

↓

Forwards the packet.

---

# Routing Table

A Router maintains a Routing Table containing:

- Destination Network
- Next Hop
- Interface
- Metric

---

# Features

- Layer 3 Device
- Uses IP Addresses
- Connects Different Networks
- Supports Routing Protocols

---

# Advantages

- Internet connectivity
- Connects multiple networks
- Selects the best route
- Supports large networks

---

# Disadvantages

- More expensive than switches
- More complex configuration
- Slightly slower than switching due to routing decisions

---
