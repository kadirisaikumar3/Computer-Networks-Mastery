# 🌐 Chapter 10 – HTTP / HTTPS

## 📖 Overview

**HTTP (HyperText Transfer Protocol)** is the foundation of communication on the World Wide Web.

Whenever you open a website, send a form, call a REST API, or download a file, HTTP is used.

**HTTPS (HyperText Transfer Protocol Secure)** is the secure version of HTTP. It encrypts communication using SSL/TLS to protect data from attackers.
