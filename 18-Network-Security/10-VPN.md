# VPN (Virtual Private Network)

## What is a VPN?

A **Virtual Private Network (VPN)** is a technology that creates a **secure, encrypted tunnel** between a user's device and a remote network over the Internet.

It protects data from being intercepted by attackers.

---

# Why is VPN Needed?

Without a VPN:

- Data travels over the Internet and may be intercepted.
- Public Wi-Fi networks are unsafe.
- Remote employees cannot securely access company resources.

A VPN solves these problems.

---


---

Step 2

```

```

---

Step 3

```
Encrypted Data

↓

Internet

↓

```

---

Step 4

VPN Server decrypts the data and forwards it.

```
VPN Server

↓

Company Network
```

# Types of VPN

### Remote Access VPN

Used by employees working remotely.

---

### Site-to-Site VPN

Connects two office networks securely.

Example:

```
Branch Office

↓

VPN Tunnel

↓

Head Office
```

---

# Advantages

- Secure Communication
- Encrypts Internet Traffic
- Protects Public Wi-Fi Usage
- Enables Remote Work
- Hides Public IP Address

---

# Disadvantages

- Slightly Slower Internet
- VPN Server Costs
- Configuration Complexity

---

# Real-World Example
