# Authentication

## What is Authentication?

**Authentication** is the process of verifying the identity of a user, device, or application before granting access.

It answers the question:

```
Who are you?
```

---

# Why is Authentication Needed?

Authentication prevents unauthorized users from accessing protected systems.

Without authentication:

- Anyone could access confidential information.
- Systems would be vulnerable to attacks.

---

# Authentication Factors

Authentication is commonly based on one or more of these factors:

### 1. Something You Know

Examples:

- Password
- PIN
- Security Question

---

### 2. Something You Have

Examples:

- Mobile Phone
- OTP
- Smart Card
- Security Token

---

### 3. Something You Are

Examples:

- Fingerprint
- Face Recognition
- Iris Scan
- Voice Recognition

---

# Multi-Factor Authentication (MFA)

MFA uses two or more authentication factors.

Example:

```
Password

+

OTP

+

Fingerprint
```

This provides stronger security than a password alone.

---
