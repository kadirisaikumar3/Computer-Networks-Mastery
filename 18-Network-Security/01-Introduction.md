# Introduction

## What is Network Security?

**Network Security** is the practice of protecting computer networks, devices, applications, and data from unauthorized access, misuse, cyber attacks, and data breaches.

It combines technologies, policies, and security practices to ensure that communication over a network remains secure.

---

# Simple Definition

Network Security protects data while it is being transmitted and stored.

Its goal is to prevent attackers from stealing, modifying, or destroying information.

---

# Why is Network Security Important?

Without Network Security:

- Hackers can steal sensitive data.
- Malware can infect systems.
- Unauthorized users may gain access.
- Services can become unavailable.
- Financial and reputation loss may occur.

---

# Objectives of Network Security

- Protect Confidential Information
- Prevent Unauthorized Access
- Ensure Data Integrity
- Maintain Service Availability
- Detect and Prevent Attacks

---

# Components of Network Security

- Firewalls
- Antivirus
- Intrusion Detection Systems (IDS)
- Intrusion Prevention Systems (IPS)
- VPN
- Encryption
- Authentication
- Access Control
