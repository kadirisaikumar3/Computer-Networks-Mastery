# CIA Triad

## What is the CIA Triad?

The **CIA Triad** is the foundation of Information Security.

It consists of three core security principles:

- Confidentiality
- Integrity
- Availability

Every secure system aims to achieve these three goals.

---

# 1. Confidentiality

Confidentiality ensures that only authorized users can access information.

Methods used:

- Passwords
- Encryption
- Multi-Factor Authentication (MFA)
- Access Control

Example:

Only employees can access company payroll data.

---

# 2. Integrity

Integrity ensures that data is not modified without authorization.

Methods used:

- Hashing
- Digital Signatures
- Checksums

Example:

A downloaded file should remain unchanged.

---

# 3. Availability

Availability ensures that systems and data are accessible whenever authorized users need them.

Methods used:

- Backups
- Redundant Servers
- Load Balancers
- Disaster Recovery

Example:

Google Search remains available even during heavy traffic.
