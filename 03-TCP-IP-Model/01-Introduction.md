# Introduction

## What is the TCP/IP Model?

The **TCP/IP (Transmission Control Protocol/Internet Protocol) Model** is the standard networking model used for communication over the Internet.

It defines how data is transmitted from one device to another across interconnected networks.

Unlike the OSI Model, TCP/IP is a practical model implemented in real-world computer networks.

---

# History

The TCP/IP Model was developed by the **U.S. Department of Defense (DoD)** to enable reliable communication between different computer systems.

Today, it is the foundation of the Internet.

---

# Objectives

- Standardize Internet communication
- Enable communication between different networks
- Provide reliable data transmission
- Support scalable and interoperable networking

---

# Number of Layers

The TCP/IP Model contains **4 Layers**.

```
Application

↓

Transport

↓

Internet

↓

Network Access
```

---

# Advantages

- Used worldwide
- Highly scalable
- Reliable
- Open standard
- Supports heterogeneous networks

---

# Limitations

- Less detailed than the OSI Model
- Does not clearly separate Session and Presentation functions
