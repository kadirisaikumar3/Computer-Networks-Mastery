# 🌐 TCP/IP Model

The **TCP/IP (Transmission Control Protocol / Internet Protocol) Model** is the practical networking model used on the Internet.

Unlike the OSI Model, which is a conceptual reference model, the TCP/IP Model is implemented in real-world networks.

It defines how data is transmitted between devices over a network.

# 🎯 Why Learn TCP/IP?

The Internet runs on the TCP/IP Model.

Every website, email, mobile app, and cloud application uses TCP/IP for communication.

Understanding this model is essential for Software Engineers and is frequently tested in interviews.
