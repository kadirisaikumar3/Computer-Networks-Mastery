# TCP/IP Model Cheat Sheet

## TCP/IP Layers

```
Application

↓

Transport

↓

Internet

↓

Network Access
```

---

## Data Units

| Layer | Data Unit |
|--------|-----------|
| Application | Data |
| Transport | Segment |
| Internet | Packet |
| Network Access | Frame / Bits |

---

## Common Protocols

### Application Layer

- HTTP
- HTTPS
- FTP
- SMTP
- DNS
- DHCP

---

### Transport Layer

- TCP
- UDP

---

### Internet Layer

- IPv4
- IPv6
- ICMP
- ARP

---

### Network Access Layer

- Ethernet
- Wi-Fi
- Bluetooth

---
