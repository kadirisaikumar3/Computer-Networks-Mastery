# Google Interview Corner

This section contains the most frequently asked TCP/IP Model interview questions.

---

# Basic Questions

## What is the TCP/IP Model?

A practical networking model used for communication over the Internet.

---

## How many layers are there?

```
4
```

---

## Name the four layers.

- Application
- Transport
- Internet
- Network Access

---

## Which model is used on the Internet?

TCP/IP Model.

---

# Layer-Based Questions

## Which layer provides services to user applications?

Application Layer.

---

## Which layer provides reliable communication?

Transport Layer.

---

## Which layer performs routing?

Internet Layer.

---

## Which layer uses MAC addresses?

Network Access Layer.

---

# Protocol Questions

## Difference between TCP and UDP.

| TCP | UDP |
|------|------|
| Reliable | Faster |
| Connection-Oriented | Connectionless |
| Ordered Delivery | No Ordering |

---

## Which protocol converts domain names into IP addresses?

DNS.

---

## Which protocol assigns IP addresses automatically?

DHCP.

---

## Which protocol is used for secure web browsing?

HTTPS.

---

## Which protocol is used for file transfer?

FTP.

---

# Scenario-Based Questions

## Website opens slowly. Which layer could be responsible?

Transport Layer or Internet Layer.

---

## Unable to resolve a website name.

Likely a DNS issue.

---

## Router is not forwarding packets.

Internet Layer.

---

## Network cable is disconnected.

Network Access Layer.
