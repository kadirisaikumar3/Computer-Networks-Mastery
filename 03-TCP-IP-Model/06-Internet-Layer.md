# Internet Layer

## Overview

The **Internet Layer** is responsible for logical addressing and routing packets between different networks.

It determines the best path for packets to travel from the source to the destination.

---

# Responsibilities

- Logical Addressing
- Routing
- Packet Forwarding
- Fragmentation
- Path Selection
