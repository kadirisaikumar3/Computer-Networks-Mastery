# Common Mistakes

---

# Mistake 1

Thinking TCP/IP has 7 layers.

✅ Correct

TCP/IP has **4 layers**.

---

# Mistake 2

Confusing the Internet Layer with the Network Access Layer.

✅ Correct

Internet Layer

- Routing
- IP Address
- Packets

Network Access Layer

- MAC Address
- Frames
- Physical Transmission

---

# Mistake 3

Thinking TCP belongs to the Internet Layer.

✅ Correct

TCP belongs to the **Transport Layer**.

---

# Mistake 4

Confusing HTTP and HTTPS.

HTTP

- Port 80
- No Encryption

HTTPS

- Port 443
- Uses TLS Encryption

---

# Mistake 5

Thinking UDP is always better because it is faster.

✅ Correct

Use TCP when reliability is important.

Use UDP when speed is more important.

---

# Mistake 6

Confusing DNS and DHCP.

DNS

- Domain → IP

DHCP

- Assigns IP Address

---

# Interview Tip

Remember every layer with:

- Function
- Protocols
- Data Unit
- Real-world Examples
