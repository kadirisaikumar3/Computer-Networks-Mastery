# Frequently Asked Questions (FAQs)

## What is the TCP/IP Model?

The TCP/IP Model is the networking model used for communication over the Internet.

---

## How many layers are in the TCP/IP Model?

```
4
```

- Application
- Transport
- Internet
- Network Access

---

## Which model is used on the Internet?

TCP/IP Model.

---

## Which layer performs routing?

Internet Layer.

---

## Which layer provides reliable communication?

Transport Layer.

---

## Which layer communicates with user applications?

Application Layer.

---

## Which layer uses MAC addresses?

Network Access Layer.

---

## Which protocol is used for secure web browsing?

HTTPS.

---

## Which protocol converts domain names into IP addresses?

DNS.

---

## Which protocol automatically assigns IP addresses?

DHCP.

---

## Which protocol is faster?

UDP.

---

## Which protocol is reliable?

TCP.
