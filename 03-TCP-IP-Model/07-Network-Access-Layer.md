# Network Access Layer

## Overview

The **Network Access Layer** is the lowest layer of the TCP/IP Model.

It combines the responsibilities of the **Physical Layer** and the **Data Link Layer** of the OSI Model.

This layer is responsible for transmitting data over the physical network.

---

# Responsibilities

- Physical Transmission
- Frame Creation
- MAC Addressing
- Error Detection
- Access to Physical Media
