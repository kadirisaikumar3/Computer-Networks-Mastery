# Google Interview Corner

This section contains the most frequently asked IP Addressing interview questions.

---

# Basic Questions

## What is an IP Address?

A unique logical address assigned to a device on a network.

---

## Difference between IPv4 and IPv6.

| IPv4 | IPv6 |
|------|------|
| 32 Bits | 128 Bits |
| 4 Octets | 8 Groups |
| Dot Notation | Colon Notation |

---

## Difference between Public IP and Private IP.

| Public | Private |
|--------|---------|
| Internet | LAN |
| ISP Assigned | Router Assigned |

---

## What is a Subnet Mask?

A subnet mask separates the Network ID and Host ID.

---

## What is CIDR?

A classless addressing method that uses prefix notation.

---

# Intermediate Questions

## Why was IPv6 introduced?

To solve IPv4 address exhaustion.

---

## What is NAT?

Network Address Translation allows multiple private IP addresses to share a single public IP.

---

## What is APIPA?

Automatic Private IP Addressing when DHCP is unavailable.

---

## What is the Loopback Address?

```
127.0.0.1
```

Used for testing the local machine.

---

# Scenario-Based Questions

## A system gets a 169.254.x.x address. Why?

The DHCP server is unavailable.

---

## Why can't two devices have the same IP address?

It causes an IP address conflict.

---

## Which address identifies the network?

The Network Address.

---

## Which address sends data to all hosts?

The Broadcast Address.
