# 🚀 Chapter 4 – IP Addressing

## 📖 Overview

IP Addressing is the foundation of modern computer networks. Every device connected to a network requires a unique IP address for identification and communication.

This chapter covers IPv4, IPv6, IP Classes, Public vs Private IPs, Subnet Masks, CIDR notation, Special IP Addresses, practical examples, hands-on labs, and Google interview questions.
