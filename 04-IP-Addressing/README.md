# 🌐 IP Addressing

An **IP Address (Internet Protocol Address)** is a unique numerical identifier assigned to every device connected to a network.

It allows devices to communicate with each other over local networks and the Internet.

Without an IP Address, computers, smartphones, servers, and other devices cannot send or receive data.
