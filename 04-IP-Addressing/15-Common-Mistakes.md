# Common Mistakes

---

# Mistake 1

Thinking MAC Address and IP Address are the same.

✅ Correct

MAC Address

- Physical Address
- Layer 2

IP Address

- Logical Address
- Layer 3

---

# Mistake 2

Confusing Public IP and Private IP.

Public IP

- Internet

Private IP

- LAN

---

# Mistake 3

Thinking IPv6 uses dotted decimal notation.

✅ Correct

IPv4 → Dots (.)

IPv6 → Colons (:)

---

# Mistake 4

Thinking 127.0.0.1 is a Public IP.

✅ Correct

127.0.0.1 is the **Loopback Address**.

---

# Mistake 5

Thinking NAT increases the number of IPv4 addresses.

✅ Correct

NAT helps conserve public IPv4 addresses by allowing multiple private devices to share one public IP.

---

# Mistake 6

Confusing Network Address and Broadcast Address.

Network Address

- Identifies the network.

Broadcast Address

- Sends packets to every device.

---

# Interview Tip

Always remember:

- Address Type
- Purpose
- Example
- Layer
