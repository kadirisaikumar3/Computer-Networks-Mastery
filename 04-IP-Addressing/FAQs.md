# Frequently Asked Questions (FAQs)

## 1. What is an IP Address?

An IP Address uniquely identifies a device on a network.

---

## 2. Why is IPv6 needed?

IPv4 has a limited address space. IPv6 provides a much larger number of addresses.

---

## 3. Can two devices have the same IP Address?

Not on the same network. It causes an IP Address conflict.

---

## 4. What is NAT?

Network Address Translation allows multiple devices with Private IP Addresses to share one Public IP Address.

---

## 5. Which IP version is mostly used today?

IPv4 is still widely used, while IPv6 adoption is increasing.

---

## 6. What is DHCP?

DHCP automatically assigns IP Addresses to devices on a network.

---

## 7. What is the difference between Public and Private IP Addresses?

Public IPs are Internet-accessible. Private IPs are used only within local networks.

---

## 8. Does every website have a Public IP Address?

Yes. Websites are hosted on servers with Public IP Addresses.

---

## 9. Why can't we use only Public IP Addresses?

There are not enough IPv4 Public IP Addresses for every device. Private IPs and NAT solve this problem.

---

## 10. Is IP Addressing important for Software Engineers?

Yes. Understanding IP Addressing helps with backend development, cloud computing, system design, networking, and debugging.
