# Hands-On Lab

## Objective

Learn IP Addressing using basic networking commands.

---

# Lab 1 – View Your IP Address

Windows

```cmd
ipconfig
```

Linux

```bash
ip addr
```

Observe

- IPv4 Address
- IPv6 Address
- Subnet Mask
- Default Gateway

---

# Lab 2 – Display Detailed Configuration

```cmd
ipconfig /all
```

Observe

- MAC Address
- DHCP Status
- DNS Server

---

# Lab 3 – Test Connectivity

```cmd
ping google.com
```

Observe

- IP Address
- Response Time
- Packet Loss

---

# Lab 4 – Display Routing Table

```cmd
route print
```

Observe

- Default Route
- Network Routes
- Gateway

---

# Lab 5 – DNS Lookup

```cmd
nslookup google.com
```

Observe

- DNS Server
- IP Address Returned

---

# Lab 6 – Trace Route

```cmd
tracert google.com
```

Observe

- Number of Hops
- Routers Traversed

---

# Practice Activity

Identify on your computer:

- MAC Address
