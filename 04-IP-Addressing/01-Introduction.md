# Introduction

## What is an IP Address?

An **IP (Internet Protocol) Address** is a unique numerical identifier assigned to every device connected to a network.

It allows devices to identify each other and communicate over a network or the Internet.

---

# Why Do We Need IP Addresses?

Without IP addresses:

- Devices cannot communicate.
- Routers cannot forward packets.
- The Internet cannot function.

Every device connected to a network requires a unique IP address.

---

# Functions of an IP Address

- Identifies a device
- Identifies the network
- Enables routing
- Enables end-to-end communication

---

# Types of IP Addresses

- IPv4
- IPv6

---

# Components of an IP Address

```
Network ID

+

Host ID
```

---

# Example

```
192.168.1.10
```

Network ID:

```
192.168.1
```

Host ID:

```
10
```

# Advantages

- Unique identification

- Internet communication
- Device-to-device communication
