# IPv4 vs IPv6

## Overview

IPv4 and IPv6 are two versions of the Internet Protocol.

IPv4 is the most widely used today, while IPv6 was introduced to overcome the limitations of IPv4.

---

# Comparison

| Feature | IPv4 | IPv6 |
|----------|------|-------|
| Address Length | 32 Bits | 128 Bits |
| Address Format | Decimal | Hexadecimal |
| Number of Addresses | ~4.3 Billion | 340 Undecillion |
| Address Separator | Dot (.) | Colon (:) |
| NAT Required | Yes | No |
| Header Size | Variable | Fixed |
| Configuration | Manual/DHCP | Auto + DHCPv6 |
| Security | Optional | Built-in IPsec Support |

---

# IPv4 Example

```
192.168.1.10
```

---

# IPv6 Example

```
2001:db8::1
```

---

# Advantages of IPv6

- Huge address space
- Better routing
- Faster processing
- Built-in security
- No NAT

---

# Why IPv6?

IPv4 addresses are nearly exhausted.

IPv6 solves this problem by providing a much larger address space.
