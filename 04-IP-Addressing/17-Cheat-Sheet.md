# IP Addressing Cheat Sheet

## IPv4

- 32 Bits
- 4 Octets
- Dot Notation

---

## IPv6

- 128 Bits
- 8 Groups
- Colon Notation

---

## Private IP Ranges

| Range | CIDR |
|--------|------|
| 10.0.0.0 – 10.255.255.255 | /8 |
| 172.16.0.0 – 172.31.255.255 | /12 |
| 192.168.0.0 – 192.168.255.255 | /16 |

---

## Special Addresses

- 127.0.0.1 → Loopback
- 169.254.x.x → APIPA
- 0.0.0.0 → Default Route
- 255.255.255.255 → Broadcast
