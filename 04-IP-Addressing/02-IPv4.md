# IPv4

## Overview

**IPv4 (Internet Protocol Version 4)** is the most widely used version of the Internet Protocol.

It uses **32-bit addresses**, allowing approximately **4.3 billion unique addresses**.

---

# IPv4 Format

```
192.168.1.10
```

It consists of **4 octets** separated by dots.

Example:

```
192 . 168 . 1 . 10
```

Each octet ranges from:

```
0 – 255
```
