# IPv6

## Overview

**IPv6 (Internet Protocol Version 6)** is the successor to IPv4.

It uses **128-bit addresses**, providing an extremely large address space.

---

# IPv6 Format

Example:

```
2001:0db8:85a3:0000:0000:8a2e:0370:7334
```

It consists of **8 groups** separated by colons.

---

# Address Size

```
128 Bits

=

16 Bytes
```

---

# Features

- Huge address space
- Better security support
- Faster routing
- No NAT required
- Auto-configuration

---

# Advantages

- Virtually unlimited addresses
- Improved performance
- Better scalability
- Built-in IPsec support

---

# IPv6 Address Compression

Leading zeros can be removed.

Example:

```
2001:0db8:0000:0000:0000:0000:0000:0001

↓

2001:db8::1
```
